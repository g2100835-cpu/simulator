社会理论人生模拟器

同一段人生，在不同的理论透镜下呈现截然不同的命运。
选择一位社会学家，用他的眼睛活一遍。
你会发现：决定命运的，也许不是你活着的方式，而是谁在注视你。

选择你的理论透镜

不同的理论框架将决定你看到怎样的人生

卡尔·马克思
历史唯物主义 · 阶级斗争
一切社会的历史都是阶级斗争的历史。你的一生，将由你在生产关系中的位置来定义——剥削或被剥削，异化或觉醒。
核心指标：阶级意识 / 异化程度 / 物质条件
皮埃尔·布迪厄
场域理论 · 资本转换 · 惯习
社会是一张资本博弈的牌桌。经济资本、文化资本、社会资本与符号资本之间的微妙转换，将决定你是上位者还是区隔对象。
核心指标：经济资本 / 文化资本 / 社会资本 / 符号资本
欧文·戈夫曼
拟剧论 · 印象管理 · 污名
人生如戏，你我皆为台上演员。每一刻你都在管理他人对你的印象——前台表演、后台准备，以及那些你拼命想藏起来的污名。
核心指标：社会面子 / 污名压力 / 角色库
塔尔科特·帕森斯
结构功能主义 · AGIL · 社会化
社会是一个自我维系的功能系统。你的一生，就是被社会化的过程——内化规范、扮演角色、维持均衡。偏离？系统自有修正机制。
核心指标：系统整合 / 角色符合 / 价值内化
米歇尔·福柯
权力/知识 · 规训 · 自我技术
权力无处不在，不仅压迫你，更塑造你。从学校的铃声到医院病历，从自我审视到坦白欲望——你是被规训出来的主体，还是逃逸者？
核心指标：规训程度 / 抵抗意识 / 主体化
安东尼·吉登斯
结构化理论 · 反思性 · 生活政治
结构与行动不是对立的两极，而是同一枚硬币的两面。你被结构塑造，但你的每一次行动也在改写结构。反思性是你的武器。
核心指标：能动性 / 结构约束 / 反思性
